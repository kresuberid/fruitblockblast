## ğŸ“² Cara Instal Game (Android/iOS/PC)

Game ini menggunakan teknologi **PWA (Progressive Web App)**, yang memungkinkan Anda menginstalnya seperti aplikasi native namun jauh lebih ringan.

1.  **Buka Game** di browser (Chrome/Safari).
2.  Tunggu loading selesai.
3.  Klik tombol **"INSTAL APLIKASI"** (ikon Download) yang muncul di layar utama.
4.  Jika tombol tidak muncul:
    *   **Android (Chrome):** Klik titik tiga di pojok kanan atas -> Pilih "Tambahkan ke Layar Utama" atau "Instal Aplikasi".
    *   **iOS (Safari):** Klik tombol Share (kotak dengan panah ke atas) -> Pilih "Add to Home Screen".
5.  Game akan terinstal dan bisa dimainkan secara **OFFLINE**!

> **Catatan:** Saat pertama kali dibuka, game akan meminta izin untuk mengirimkan **Notifikasi** (untuk info skor tertinggi).
